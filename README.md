# E-commerce Product Rating Prediction

## 📌 Problem Statement
Predict product ratings based on features like price, category, etc.

## 🎯 Objective
Build a machine learning model to predict user ratings.

## 📂 Dataset
- train.csv
- test.csv

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## 🚀 How it Works
1. Data preprocessing
2. Feature selection
3. Model training
4. Prediction

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run:
   python main.py

## 📊 Output
- Linear Regression MAE: 0.1862826064774928
- Decision Tree MAE: 0.16785083226313482
